    NavPoly

    A class that represents a navigable polygon with a navmesh. It is built on top of a Polygon. It implements the properties and fields that javascript-astar needs - weight, toString, isWall and getCost. See GPS test from astar repo for structure: https://github.com/bgrins/javascript-astar/blob/master/test/tests.js
